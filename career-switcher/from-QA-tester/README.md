# 1. Kenapa QA / Tester Sangat Cocok Masuk Cybersecurity?
Background QA sangat kuat untuk masuk security karena:

- Sudah terbiasa membuat test case ‚Üí mudah beralih ke exploit chain  
- Terbiasa melakukan exploratory testing ‚Üí mirip dengan vulnerability discovery  
- Memahami bagaimana aplikasi bekerja ‚Üí modal besar untuk AppSec  
- Automation QA dapat menulis script ‚Üí berguna untuk automation exploitation  
- Familiar dengan API testing ‚Üí cocok untuk API security testing  
- Terbiasa debugging & reproduksi bug ‚Üí mirip proses security triage  

Inilah kenapa banyak pentester atau AppSec engineer yang berasal dari QA.
