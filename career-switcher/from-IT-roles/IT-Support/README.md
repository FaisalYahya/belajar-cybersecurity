# 5. Home Lab Guide

## SIEM Lab (Wazuh + Sysmon)  
1. Install Wazuh  
2. Install Sysmon  
3. Forward logs via Winlogbeat  
4. Build dashboard  
5. Create detection rules  

**References**  
- Wazuh Docs: https://documentation.wazuh.com  
- Sysmon Config: https://github.com/SwiftOnSecurity/sysmon-config  

---

## Sysmon Event ID Penting  
- 1 (Process creation)  
- 3 (Network connection)  
- 7 (Image loaded)  
- 11 (File created)  

---

## Vulnerability Scan Lab  
1. Install Nessus Essentials  
2. Run host scan  
3. Focus on:  
   - SMBv1  
   - Open ports  
   - Missing patches
