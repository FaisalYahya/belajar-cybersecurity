# 1. Kenapa Non‚ÄëIT Bisa Masuk Cybersecurity?

Cybersecurity bukan hanya soal hacking. Banyak role yang:

- Tidak perlu background IT  
- Tidak perlu coding  
- Fokus pada analisis, dokumentasi, dan pemahaman risiko  
- Bisa dipelajari dari nol dengan roadmap yang tepat  

Cybersecurity membutuhkan:

- **Analytical thinking** ‚Üí dimiliki oleh banyak non‚ÄëIT roles  
- **Problem solving**  
- **Research & documentation**  
- **Attention to detail**  

Jika kamu punya salah satu dari skill di atas ‚Üí kamu sudah punya pondasi kuat.

# 5. Home Lab Guide (Sangat Ramah Pemula)

## A. SIEM Beginner Lab (Wazuh)
1. Install Wazuh Manager (tutorial tersedia di website)  
2. Install Sysmon di Windows  
3. Kirim log ke Wazuh  
4. Cari event mencurigakan:  
   - login gagal  
   - powershell execution  
   - service creation  

---

## B. Simple Threat Analysis Lab (Non-Coding)
Gunakan:

- VirusTotal  
- HybridAnalysis  
- AbuseIPDB  

Tugas:

- Cek hash malware  
- Cek domain/URL suspicious  
- Tulis report sederhana  

---

## C. GRC Portfolio Lab (Non-Technical)
Buat 3 dokumen:

- Access Control Policy  
- Incident Response Plan  
- Data Classification Policy
