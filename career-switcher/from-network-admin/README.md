# 1. Kenapa Network Admin Mudah Masuk Cybersecurity?
Latar belakang jaringan sangat kuat dan sangat dicari di dunia security, karena:

- Paham routing, switching, firewall, NAT, VLAN  
- Paham monitoring traffic  
- Sudah terbiasa dengan CLI devices (Cisco, Mikrotik, Juniper)  
- Paham network troubleshooting  
- Paham protocol-level behavior (TCP/IP, DNS, DHCP, BGP)  

Ini sangat relevan dengan area security seperti:

- Network security  
- Intrusion detection  
- Threat hunting  
- Incident response  
- Cloud security  
- Zero Trust networking

# 5. Home Lab Guide (Praktikal)

## A. **Network Security Lab**
Setup:
- PFsense or OPNsense firewall  
- Suricata IDS/IPS  
- Kali Linux  
- Windows + Sysmon  

Goals:
- Block malicious traffic  
- Analyze alerts  
- Create IDS rules  

---

## B. **Packet Analysis Lab**
Gunakan Wireshark untuk analisis:
- DNS tunneling  
- Beaconing  
- Suspicious PowerShell  
- SMB brute force  

---

## C. **Security Onion Lab**
Gunakan:
- Suricata  
- Zeek  
- ElasticStack  

Goals:
- End-to-end network monitoring  
- Correlation across alerts
