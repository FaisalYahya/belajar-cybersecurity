Folder ini berisi modul pembelajaran cybersecurity untuk pemula.  
Tujuannya memandu belajar bertahap, dari dasar IT hingga cloud security.  
Setiap modul ringkas, praktis, dan siap dipakai di lab pribadi.

# Daftar Modul

1. [modul-01-dasar-it](./modul-01-dasar-it/README.md) — Fondasi IT untuk jalur cybersecurity: konsep inti, alat, dan praktik.
2. [modul-02-dasar-jaringan](./modul-02-dasar-jaringan/README.md) — Dasar jaringan: OSI, TCP/IP, subnetting, VLAN, routing, NAT, dan DNS.
3. [modul-03-linux-dasar](./modul-03-linux-dasar/README.md) — Dasar Linux: CLI, filesystem, permission, proses, layanan, paket, dan networking.
4. [modul-04-pengenalan-security](./modul-04-pengenalan-security/README.md) — Pengantar security: perbedaan cybersecurity dan InfoSec, prinsip, kontrol, dan incident response.
5. [modul-05-web-security-dasar](./modul-05-web-security-dasar/README.md) — Dasar keamanan web: HTTP, auth, XSS, injection, CORS, CSP, dan WAF.
6. [modul-06-cloud-security-dasar](./modul-06-cloud-security-dasar/README.md) — Dasar cloud security: IAM, jaringan virtual, KMS, logging, IaC, dan guardrails.
